## What this repository contains

- **Mini React projects**  
  Small apps like counters, lottery game, Ludo board, todo list, weather UI, pricing cards, etc.  
  These were built to practice:
  - `useState`
  - basic `useEffect`
  - props and component structure
  - handling arrays and objects in state

- **React landing pages & UI practice**  
  Simple landing pages and UI-only components built to get comfortable with layout and styling in React.

- **Full-stack practice projects (MERN-style)**  
  Some folders contain React apps connected to a backend (Node / MongoDB) created while learning how frontend and backend talk to each other.

- **Experiments & refactors**  
  Old code revisited later to fix bugs, update dependencies, or improve state handling.

## How to run a project (general)

Each project usually lives in its own folder.

Basic steps for most React apps:

```bash
npm install
npm run dev
```

## What I learned from these projects

- Why React state must be updated immutably (objects & arrays)
- How small components make debugging easier
- Common beginner mistakes with state updates and re-renders
- How React apps grow messy without structure (and why refactoring matters)
- How frontend fits into a full-stack application
Revisiting these projects later helped me clearly see my progress.
